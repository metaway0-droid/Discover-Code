# Discover Code 🚀 (2026)

**Discover Code**는 GitHub API를 활용하여 전 세계의 인기 있는 오픈소스 프로젝트를 발견하고 분석하는 최첨단 AI 기반 웹 어플리케이션입니다. 2026년의 최신 웹 기술과 디자인 트렌드를 반영하여 제작되었습니다.

---

## ✨ 주요 기능 (Key Features)

- **🔥 Rising Now**: 최근 7일 이내 생성된 인기 프로젝트를 실시간으로 탐색.
- **🖼 Premium Grid Cards**: 정보를 직관적이고 아름답게 배치하는 3열 그리드 카드 시스템.
- **🇰🇷 Auto-Translation**: 모든 프로젝트 설명을 한국어로 자동 번역하여 정보 격차 해소.
- **🤖 Smart AI Summary**: 긴 README 본문을 핵심만 골라 한국어로 요약해주는 지능형 분석 모달.
- **💖 Persistent Favorites**: `localStorage`를 활용해 나만의 보관함을 유지 (새로고침 후에도 유지).
- **📊 Tech Insights**: 검색된 결과의 기술 스택 분포를 실시간 통계로 시각화.

---

## 🛠 기술 스택 (Tech Stack)

- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS v4, Native CSS Nesting, Glassmorphism 2.0
- **Animation**: Framer Motion, Scroll-driven Animations
- **Icons**: Lucide React
- **Markdown**: ReactMarkdown, rehype-raw (HTML Support)

---

## 🚀 시작하기 (Quick Start)

1. **의존성 설치:**
   ```bash
   npm install
   ```

2. **로컬 개발 서버 실행:**
   ```bash
   npm run dev
   ```
